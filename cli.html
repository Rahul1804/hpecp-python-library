Wait for resource to have one or more statuses.

k8s_supported_versions(output='json', major_filter=None, minor_filter=None, patch_filter=None)[source]¶

Print a list of supported k8s versions.

Parameters

  • output – how to print the output, ‘json’ or ‘text’

  • major_filter – only return versions matching major_filter

  • minor_filter – only return versions matching minor_filter

  • patch_filter – only return versions matching patch_filter

Example:

hpecp k8scluster k8s_supported_versions –major-filter 1

–minor-filter 17
