wait_for_status(k8scluster_id, status=[], timeout_secs=60)¶

Wait for cluster status.

Args:

k8scluster_id: (int) the K8S cluster ID status: (list) of type K8sClusterStatus status(es) to wait for.

Use an empty array if you want to wait for a cluster’s existence to cease.

timeout_secs: (int) how long to wait for the status(es)

Returns:

bool: True if status was found before timeout, otherwise False

Raises:

APIItemNotFoundException: if the item is not found APIException: if a generic API exception occurred
